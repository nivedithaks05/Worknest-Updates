# WorkNest

WorkNest – A platform for managing workspaces, tasks, and collaboration.

## Project Info
This project was initially created using Lovable.

## Technologies Used
- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn-ui
- Django (Backend for WorkNest)

## Running the Project Locally

Clone the repository:

git clone https://github.com/sripadsanand/vite-react-starter.git

Go into the project folder:

cd worknest

Install dependencies:

npm install

Run the development server:

npm run dev
